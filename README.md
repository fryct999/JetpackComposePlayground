# Jetpack Compose Playground

Дополнительные практические задания для закрепления навыков из курса **AndroidSprint**.

**[Курс AndroidSprint →](https://ievetrov.ru/kurs-android-razrabotchik/)**

## Структура заданий

Каждый блок заданий соответствует главе из курса AndroidSprint:

### JCP-01: Введение в декларативную верстку
Базовые компоненты, модификаторы, списки

### JCP-02: Работа с состоянием  
Кнопки, формы, валидация, таймеры

### JCP-03: Jetpack Navigation
Навигация, параметры, анимации, deep links

### JCP-04: Корутины и Flow API
StateFlow, SharedFlow, обработка ошибок

## Как пользоваться

1. Пройди соответствующий этап в AndroidSprint
2. Запусти этот проект в Android Studio
3. Выбери задание на главном экране
4. Ищи `TODO` комментарии в коде
5. Реализуй функциональность для закрепления навыков
6. Тестируй результат прямо в приложении

В каждом файле есть:
- Описание задания в шапке
- Примеры кода из соответствующего урока  
- Готовые заготовки функций
- Подробные TODO с подсказками

## Уровень сложности

**JCP-01** → Новички в Compose  
**JCP-02** → Базовые навыки состояния  
**JCP-03** → Уверенные в навигации  
**JCP-04** → Продвинутые (корутины + Flow)
